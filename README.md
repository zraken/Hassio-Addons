## Installation
### Option 1
1. Click on the following button:

    [![Open your Home Assistant instance and show the add app repository dialog with a specific repository URL pre-filled.](https://my.home-assistant.io/badges/supervisor_add_addon_repository.svg)](https://my.home-assistant.io/redirect/supervisor_add_addon_repository/?repository_url=https%3A%2F%2Fgithub.com%2Fzraken%2FHassio-Addons)
2. Enter the URL of your Home Assistant instance if requested.
3. Click on Add to add the repository to the App store.

Each App can then be installed from the App store.

### Option 2
1. Navigate in your Home Assistant frontend to **Configuration** -> **Apps** (Formerly Add-ons)
2. Click on **Install App** in the bottom right corner
3. Click on the 3 dots menu in the top right corner and select **Repositories**
4. Add this URL as an additional repository: ```https://github.com/zraken/Hassio-Addons```

Each App can then be installed from the App store
